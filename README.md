# foodpan
## 🍔 Foodpanda Data Analysis Project

## 📊 Project Title

**Business Intelligence & Customer Insights Analysis – Foodpanda Dataset**

---

## 📌 Project Overview

This project focuses on analyzing a real-world Foodpanda food delivery dataset containing customer, restaurant, and transaction data across multiple cities in Pakistan.

The goal is to transform raw data into actionable business insights that can support strategic decision-making in areas such as customer retention, operational efficiency, and revenue growth.

---

## 🎯 Objectives

* Clean and prepare raw data for analysis
* Build a structured data model
* Create key performance indicators (KPIs)
* Develop interactive dashboards
* Generate insights to improve business performance
* Communicate findings clearly and effectively

---

## 🗂️ Dataset Description

The dataset includes the following key components:

### 👤 Customer Information

* Customer ID
* Location (City)
* Order history
* Frequency of orders

### 🍽️ Restaurant Information

* Restaurant ID
* Cuisine type
* Ratings
* Delivery performance

### 📦 Order / Transaction Data

* Order ID
* Order date & time
* Order value
* Payment method
* Delivery status

### 🚚 Service Performance

* Delivery time
* Order completion rate
* Delays and cancellations

## 📈 Key KPIs Created

* Total Orders
* Total Revenue
* Average Order Value (AOV)
* Customer Retention Rate
* Delivery Time (Average)
* Order Cancellation Rate
* Top Performing Restaurants

---

## 📊 Dashboard Features

* Sales and revenue trends over time
* Customer segmentation insights
* City-wise performance analysis
* Restaurant ranking and ratings
* Delivery efficiency metrics

---

## 🔍 Key Insights

* High-performing cities contribute the majority of revenue
* Repeat customers significantly increase overall sales
* Delivery delays impact customer satisfaction and retention
* Certain cuisines perform better in specific locations

---

## 💡 Recommendations

* Improve delivery logistics in low-performing areas
* Implement loyalty programs to retain customers
* Partner with top-performing restaurants for promotions
* Optimize delivery routes to reduce delays
